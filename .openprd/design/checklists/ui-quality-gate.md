# UI Quality Gate

## P0

- 已写清本次审美主张：用途、受众、气质端点、约束和用户会记住的一件事
- 关键事实已核实，或明确标注 `待确认`
- 主题已锁定，不存在临时发明的颜色体系
- 字体、色彩、动效、表面/背景和构图选择能服务同一个审美主张
- 真实图片是否必需已经判断
- 大界面任务若无参考方向，已完成 3 个异源方向
- 已用 `anti-slop.md` 排除默认紫白渐变、通用字体栈、白底卡片堆叠和无语境装饰

## P1

- 页面主次明确
- 同一页面的密度一致
- 按钮、卡片、表格、表单大小关系自然
- 长文本没有挤爆容器
- 局部视觉语言没有突然换轨
- 字体角色清楚：标题、正文、标签、数字或代码不混成一套无差别样式
- 主色、强调色、功能色和弱化背景色有分工，不是一种颜色深浅铺满全页
- 动效有一两个关键时刻，而不是散乱堆满 hover 或完全没有状态反馈

## P2

- 屏幕第一眼能知道页面是干什么的
- 未改区域没有明显视觉漂移
- 需要时已准备 `visual-prepare` 或 `visual-compare` 证据
- 视觉证据能看出选定方向的气质和记忆点，而不只是结构摆放正确
